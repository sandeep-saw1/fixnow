HTML (HyperText Markup Language) is the basic language used to create and structure web pages. It uses tags to define elements like text, links, and images, forming the foundation of every website.

CSS (Cascading Style Sheets) is a language used to style and design web pages. It controls the layout, colors, fonts, spacing, and overall appearance of HTML elements, making websites look attractive and user-friendly.

React is a JavaScript library for building fast, interactive user interfaces with reusable components. It’s popular for creating single-page applications (SPAs).

Java is a high-level, object-oriented programming language used to build web, desktop, and mobile applications. It runs on the Java Virtual Machine (JVM), making it platform-independent and reliable.

Python is a versatile, high-level language known for its readable syntax and extensive libraries. It's widely used in web development, data science, and automation due to its flexibility and ease of learning.

SQL (Structured Query Language) is the standard language for managing and manipulating relational databases. It enables users to query, insert, update, and delete data efficiently.
